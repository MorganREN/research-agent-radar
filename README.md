# research-agent-radar
An autonomous multi-agent system for continuous academic paper discovery,
analysis, and synthesis across AI and civil engineering domains.

## Launch
To launch the Research Agent Radar, run the following command:

```bash
python -m src.main_demo
```

To launch the Dashboard, run:

```bash
poetry run streamlit run src/dashboard/app.py
```

## Features
- Multi-agent architecture
- Institutional login support
- Cross-domain research analysis
- 24/7 deployment-ready
- Dashboard visualization
